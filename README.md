# reachOut-backend
An application to send sms and email to users

### Technologies
- Express 4.16.4,
- Graphql 14.1.1
- Apollo-server 2.4.8,

### Getting Started
- Clone the repository
```bash
$ git clone https://github.com/andela/hofFrontEndApp.git
```
- Navigate to the project directory
```bash
 $ cd hofFrontEndApp
```
